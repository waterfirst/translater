# Conference Helper (학회 도우미)

International Conference Real-time Translation Subtitle Web App.

## Features
- **Real-time Speech Recognition**: Supports 7 languages using Web Speech API.
- **Instant Translation**: Uses Google Translate API (Free endpoint) for immediate subtitles.
- **Camera Overlay**: Overlays subtitles on the camera feed for augmented reality feel.
- **Screenshot & Gallery**: Capture moments with subtitles and save them.
- **PWA Support**: Installable on mobile devices (Android/iOS).
- **Dark Mode UI**: Optimized for low-light conference environments.

## Supported Languages
- English (US)
- Korean (한국어)
- Japanese (日本語)
- Chinese (中文)
- German (Deutsch)
- French (Français)
- Spanish (Español)

## How to Run
1. Host these files on a secure server (HTTPS) or localhost.
   - **Note**: Camera and Microphone access requires HTTPS.
2. Open `index.html` in Chrome or Edge browser.
3. Allow Camera and Microphone permissions.

## Deployment on GitHub Pages
1. Upload all files to a GitHub repository.
2. Go to Settings -> Pages.
3. Select the branch and folder (root).
4. Save and wait for the site to be published.
